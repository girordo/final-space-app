### Como usar

```bash

# Clone o repositório
$ git clone https://github.com/girordo/final-space-app.git

# Entre na pasta
$ cd final-space-app

# Executar o projeto
$ yarn dev

# Docker build
$ docker build -t final-space .

# Docker run
$ docker run -p 80:80 final-space

```
